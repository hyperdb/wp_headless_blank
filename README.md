# wp_headless_blank

WordPressをヘッドレスCMSとして利用する際にフロントに何も表示させないためのテーマです。
`wp_headless_blank`ディレクトリをWordPressの`wp-content`ディレクトリの下に配置します。

## 補足事項

- 記事を投稿する際にslugを自動生成するファンクションを追加
    - `POST TYPE`と`POST ID`を結合した文字列からMD5ハッシュを取得してslugにしています
